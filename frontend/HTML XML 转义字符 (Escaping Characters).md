| Character  | English | Escaping  | 
| ---- | ---- | ---- |
| <  | less than  | `&lt;`  | 
| >  | greater than  | `&gt;`  | 
| &  | ampersand  | `&amp;`  | 
| "  | quote  | `&quot;`  |
| '  | apostrophe「不常用，因为在双引号里，它不不需要被转义」  | `&apos; &#39;`  |
| 空格(仅HTML，XML不需要这个东西)  | non-breakable space「不换行空格，用于禁止换行/禁止合并」  | `&nbsp;` |
